Singleplayer






You should check out the live server before thinking about singleplayer! Always up to date and supported.

Why play live instead of Singleplayer? I completely understand being concerned about your player save incase the worse happened where we went down. However, it has been state a multitude of times in the Discord that the player saves belong to the player and if God Forbid the server went down you would receive a copy of your player save with the updated copy of singleplayer. You can see more of what information we collect if you're concerned on that end using the link

Single player is the game mode that makes us super unique and we do take pride in it. The server started to focus mainly on the live server after progress slowed down completely to focus on both the singleplayer copies and the personal servers. At this point we had stopped the support for both singleplayer and personal servers.

The future is looking bright for singleplayer with the new copies coming out of the new 2.009 core for both Linux and Windows. The official support will begin again once those copies are added to the game.

The unsupported official copies of single player are still usuable but super out of date compared to 2009scape Legacy github source code.

Setting up Windows Singleplayer

  • 1: Download or clone this repository.
  • After you unpack the .zip (if you just downloaded it) or it finishes cloning (if you cloned it), open up the folder that it created.
  • In this folder there will be a run-windows.cmd, execute this to start everything up.
  • You can simply run the game with option 1.
  • When you're done, make sure to use the "exit" option in the menu to properly shut everything down.


Setting up Linux Singleplayer
  • Download or clone this repository.
  • After you unpack the .zip (if you just downloaded it) or it finishes cloning (if you cloned it), open up the folder that it created.
  • In this folder there will be a run-linux.sh, execute this to start everything up.
  • What you need to do first before you can run the game is initialize the database. You only have to do this once.
  • Once that's finished, you can safely run the game with option 1.

Frequently Asked Questions

Is there a way to make a friend join the single player version?

Yes, just open your ports : 5555 and 43595.


Is there a point of the GE?

Yes, server sided bots such a adventure bots still collect resources and sell to your ge. There is a possible way of playing the server offline while still being able to buy things. You can also sell lobster, swordfish, or shark for green dragon killing bots to buy.


Can I still play minigames?

Of Course, majority of the minigames do not require other players to play in the first place. But we have special pest control bots that will play pest control with you on the easy and intermediate boats so you can get your very own set of void armour!


How do I get the PvP and wilderness bots in Server/plugin/ai to work?

They never have worked since Arios did not program them right. Another reason we are leaving their core behind us and using our own.

Is Singleplayer actually Singleplayer or is a local network version?

It is solely singleplayer like stated earlier there are ways to open your ports up but at that point you should just fork your own personal server.
