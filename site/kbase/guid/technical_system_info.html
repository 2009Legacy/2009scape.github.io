How do I find my system information?



Finding your Graphics Card

How to locate the type of your graphics card for Windows 2000 and XP Operating Systems:
  1. Right-click on your Windows Desktop. When the dialog box comes up, select 'Properties'
  2. Click the 'Settings' tab
  3. Click the 'Advanced' button
  4. Choose the 'Adapter' tab
  5. The type of your graphics card will be displayed

How to locate the type of your graphics card for Windows Vista Operating System:
  1. Click 'Start'
  2. In the search box, type 'dxdiag'
  3. Click the 'Display' tab
  4. The driver provider, date and version is shown.

How to locate the type of your graphics card for Mac OS:
  1. Click on the blue Apple icon in the top left corner of your screen and select 'About this Mac'
  2. In the window that pops up, click the "More Info..." button
  3. Select "Graphics/Displays" option from the "Hardware" list on the left
  4. The graphics card and driver version are displayed

Finding your Graphics Card Driver

How to locate your graphics card driver version for Windows 2000 and XP Operating Systems:
  1. Right-click on your Windows Desktop. When the dialog box comes up, select 'Properties'
  2. Click the 'Settings' tab
  3. Click the 'Advanced' button
  4. Choose the 'Adapter' tab
  5. Click 'Properties' and select the 'Driver' tab
  6. The driver provider, date and version is shown.

How to locate your graphics card driver version for Windows Vista Operating System:
  1. Click 'Start'
  2. In the search box, type 'dxdiag'
  3. Click the 'Display' tab
  4. The driver provider, date and version is shown.

How to locate your graphics card driver version for Mac OS:
  1. Click on the blue Apple icon in the top left corner of your screen and select 'About this Mac'
  2. In the window that pops up, click the "More Info..." button
  3. Select "Graphics/Displays" option from the "Hardware" list on the left
  4. The graphics card and driver version are displayed

Finding your CPU, RAM and Operating System information

How to locate your CPU, RAM, OS system information for the Windows 2000 and XP Operating Systems:

To locate your system software information, as well as some information regarding your hardware, perform the following steps:
  1. Start Menu > Settings > Control Panel
  2. Select the 'System' icon
  3. On the 'General' tab there should be information about what version of Windows you are running, how much RAM you have and, sometimes, the speed of your CPU

How to locate your CPU, RAM, OS system information for the Windows Vista Operating System.

To locate your system software information, as well as some information regarding your hardware, perform the following steps:
  1. Start Menu
  2. Right-click 'Computer'
  3. Select 'Properties'

How to locate your CPU, RAM, OS system information for Mac OS:
  1. Click on the blue Apple icon in the top left corner of your screen and select 'About this Mac'
  2. In the window that pops up, you will see the Operating System version as well as the processor and memory information for your Mac

How to locate your Java version

To locate your Java software information in Windows (only if you have Sun Java installed):
  1. Start Menu > Settings > Control Panel
  2. Select the 'Java' icon
  3. Click the 'About' button to see the Java version you are running

To locate your Java software information in Mac OS (only if you have Sun Java installed):
  1. Click on the blue Apple icon in the top left corner of your screen and select 'About this Mac'
  2. In the window that pops up, click the 'More Info...' button
  3. Select 'Software'
  4. Select 'Applications', and wait for all of the applications to load
  5. Scroll down to 'Java Preferences' and 'Java Web Start'

How to locate your browser's type and version
  1. Open your browser (Internet Explorer, Firefox, etc.)
  2. Select the 'Help' menu and choose 'About'. This should display version information for your browser

We support the following browsers for both the game and the website:
  • Firefox 1.5, 2 and 3
  • Internet Explorer 6 and 7
  • Opera 9
  • Safari 2 and 3
